# 📝 Termo Clone (React Native + Expo)

Este projeto é um clone do popular jogo **Termo**, desenvolvido utilizando **React Native** com **Expo**. O objetivo é recriar a experiência do jogo original em um aplicativo mobile, com foco em performance, organização de código e boa experiência do usuário.


## 🚀 Tecnologias utilizadas

- **React Native**
- **Expo**
- **JavaScript / TypeScript**
- **Hooks (useState, useEffect)**
